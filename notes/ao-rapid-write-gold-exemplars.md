# Optional gold voice samples (Rapid Write)

If automatic tone matching still feels off on certain posts, you can add a few paragraphs **from your own published writing** that show the level and rhythm you want. The assistant uses them only as a **tone guide**—it should not repeat your sentences.

**How to use this file**

1. Paste **3–5 short paragraphs** (plain text is fine) between the two marker lines below. You can pull them from pieces you are proud of.
2. Save the file. After the next site update, new Rapid Write drafts, revisions, and optional register polish will see that block first when building style context.
3. To turn this off, delete everything between the markers (leave the markers in place) or remove the whole file.

Only text **between** the markers is sent to the assistant; notes above or below the markers are for you and stay private to this file.
